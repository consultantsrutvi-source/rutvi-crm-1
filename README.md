# Rutvi Consultants — Internal CRM

**Rutvi Consultants — Small CRM** is a lightweight internal tool to manage:
- College requirements
- Lecturer / candidate data
- Placement tracking, provisional invoice & EMI schedule
- Automatic storage into Google Sheets (via a published Apps Script Web App)
- CSV export, printable summaries, and calendar links for EMI dates

> This repository contains the front-end (GitHub Pages `index.html`) and documentation.  
> The web UI sends requests to a Google Apps Script Web App (`/exec`) which writes to your Google Sheet.
